# 42 Friends

A Chrome extension that adds a friend system floating overlay to the 42 intra.

## Installation

### Chrome

1. Download or clone this repository
2. Run `npm install` and `npm run build`
3. Open Chrome and navigate to `chrome://extensions`
4. Enable **Developer mode** (toggle in top right)
5. Click **Load unpacked** and select the `build/` folder
6. Visit https://profile.intra.42.fr to use the extension

### Firefox

1. Open Firefox and navigate to https://addons.mozilla.org/en-US/firefox/addon/42-friends/
2. Install the add-on
3. Visit https://profile.intra.42.fr to use the extension
